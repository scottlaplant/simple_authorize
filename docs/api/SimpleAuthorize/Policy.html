Overview

This class is abstract.

Subclass and override query methods to implement authorization logic

Base policy class that all other policies inherit from.

Policies encapsulate authorization logic for a specific model or resource. Each policy typically corresponds to a model class and defines methods that determine whether a user can perform specific actions.

Examples:

Basic policy implementation

class PostPolicy < ApplicationPolicy
  def update?
    user.present? && (record.user_id == user.id || user.admin?)
  end

  def destroy?
    update? # Same logic as update
  end
end

Using helper methods

class PostPolicy < ApplicationPolicy
  def update?
    owner? || admin?
  end
end

With attribute-level authorization

class PostPolicy < ApplicationPolicy
  def visible_attributes
    if admin?
      [:id, :title, :body, :user_id, :published]
    elsif owner?
      [:id, :title, :body, :published]
    else
      [:id, :title, :body]
    end
  end
end

#editable_attributes ⇒ Array<Symbol>

Returns array of attributes that the user can edit/modify.

Override this method to define which attributes can be modified by the current user. Supports action-specific variants like editable_attributes_for_create and editable_attributes_for_update.

Examples:

Basic implementation

def editable_attributes
  if admin?
    [:title, :body, :published]
  elsif owner?
    [:title, :body]
  else
    []
  end
end

Action-specific attributes

def editable_attributes_for_create
  [:title, :body] # Can't set published on create
end

def editable_attributes_for_update
  admin? ? [:title, :body, :published] : [:title, :body]
end

Returns:

  • (Array<Symbol>) —

    empty array by default (no attributes editable)

#visible_attributes ⇒ Array<Symbol>

Returns array of attributes that the user can view.

Override this method to define which attributes should be visible to the current user. Supports action-specific variants like visible_attributes_for_index and visible_attributes_for_show.

Examples:

Basic implementation

def visible_attributes
  if admin?
    [:id, :title, :body, :user_id]
  else
    [:id, :title, :body]
  end
end

Action-specific attributes

def visible_attributes_for_index
  [:id, :title] # Brief for lists
end

def visible_attributes_for_show
  [:id, :title, :body, :published] # Full for detail
end
