Overview

Exception raised when a user is not authorized to perform an action.

This error includes information about the query method, record, and policy that was used. It supports I18n for custom error messages.

Examples:

Rescuing authorization errors

rescue_from SimpleAuthorize::Controller::NotAuthorizedError, with: :user_not_authorized

def user_not_authorized(exception)
  flash[:alert] = exception.message
  redirect_to root_path
end
