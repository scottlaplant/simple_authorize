Overview

Automatic verification module to ensure all actions are authorized.

Include this module to automatically verify that every controller action calls either #authorize or #policy_scope. Helps prevent accidentally forgetting to authorize actions.

Examples:

Enable auto-verification

class ApplicationController < ActionController::Base
  include SimpleAuthorize::Controller
  include SimpleAuthorize::Controller::AutoVerify
end

Skip verification for specific actions

class PostsController < ApplicationController
  skip_authorization_check only: [:public_index]
end
