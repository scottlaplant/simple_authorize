Overview

SimpleAuthorize provides a lightweight authorization framework for Rails applications without external dependencies. It offers policy-based access control inspired by Pundit.

.configure {|Configuration| ... } ⇒ void

This method returns an undefined value.

Configure SimpleAuthorize with a block.

Yields the configuration object for modification. This is the primary way to configure SimpleAuthorize.

Examples:

SimpleAuthorize.configure do |config|
  config.default_error_message = "Access denied!"
  config.enable_policy_cache = true
end
